# codernt8-website

Static multi-page website for [codernt8.com](https://codernt8.com).

## Pages

| File | URL |
|---|---|
| `index.html` | `/` |
| `contact.html` | `/contact` |
| `risk-disclosure.html` | `/risk-disclosure` |
| `trading-platform.html` | `/trading-platform` |
| `vps.html` | `/vps` |

## Hosting

Deployed via [Netlify](https://netlify.com). Clean URLs handled by `netlify.toml` and `_redirects`.

## Editing

Edit files directly in GitHub or clone the repo. Every push to `main` auto-deploys via Netlify.
